Quick Start

To see the app do something cool, try out the location recognition plugin. This plugin will learn to recognize places that you spend time, based on building fingerprints from observable wifi base stations. To try it out, first install the application, then start the app and go into the Settings menu. Choose input plugins, and enable the Wifi Logger, and choose 1 minute for the update interval. Next, go back into the Setting menu and choose the Output Plugins settings. Enable the Location clustering plugin. Now, go back to your home screen and press and hold anywhere on the screen. This should bring up a menu that allows you to add a Widget to your home screen. Add the Location Status widget. Now, go back into the HumanSense application and start the service.

The service collects data in the background, so you can go back to your home screen (press the Home button) and use other apps. The widget on your home screen should be updated with your status (green means you're moving, read means you're stationary) and a location ID, which is UNKNOWN if you have not been stationary in a location for long enough for it to be considered a significant location, or a positive integer denoting the ID associated with the current location. If a positive integer appears, you can assign a textual label to this location by clicking on the widget. This should bring up a new dialog window that allows you to provide a name for the location. From then on, if you are in that location, the name should appear instead of the ID number.

This is a work in progress, so please let me know how it works.

Download

You can download a compiled version, ready to install, here. Included on that page is a QR code for easy installation on your phone. Note that it may not be the most up-to-date version, but it should be fairly current, and is most likely a version that has been tested a bit.
